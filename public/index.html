About

These are my solutions for the 2023 Advent of Code. I decided this year to code my solutions in JavaScript. Well, the original plan was to try and work on some visualizations in a front-end library, since that's so far from what I'm normally about, but the root of that would be coding in JavaScript, and thus far none of the visualization part has happened. In any case, coding in JavaScript meant it was only natural to run the stuff in a browser, and the original visualization plan meant I could put all the stuff in a site, so...here's the site. The front end of this may or may not improve as the month goes on, depending on how much free time I have and how much motivation I find to actually do any front-end work. In the meantime, enjoy this incredibly basic UI from an inveterate back-end coder.

You can find the source for all this at this GitHub repository, and the README file there has futher explanations and a daily log.
